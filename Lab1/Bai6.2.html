Two Column - Left Column Static

Ver 2.3

The left comlumn is absolutly positioned and is a fixed width. It won't move.

The content column, which you are reading from, is flexiable. It will automatically change shape, width can be controlled by setting appropriate margins or an actual width can be specified if required.

  • Netscape 7.1
